# 🌿 Tony's Landscaping LLC | Digital Brand Architecture

[![Status](https://img.shields.io/badge/Status-Production-success?style=for-the-badge)](https://tonyslandscapingllc.com)
[![Tech Stack](https://img.shields.io/badge/Stack-Vanilla%20JS%20|%20CSS3%20|%20EmailJS-darkgreen?style=for-the-badge)](https://github.com/DTE-84/TonysLandscaping)

**Tony's Landscaping LLC** is a premium digital experience engineered for a high-end landscape architecture firm in St. Louis. More than a simple business site, this platform is a conversion-optimized engine that bridges the gap between craftsmanship and digital discovery.

---

## ⚡ Technical Core & Features

- **Interactive Before/After Sliders**: Custom-engineered kinetic sliders that provide immersive visual proof of property transformations.
- **Project Scope Estimator**: A dynamic investment calculator that allows potential clients to self-qualify by selecting property size and project types.
- **High-Fidelity Service Galleries**: Modal-based lightbox systems for deep-dive exploration of Stone Work, Softscapes, and Retaining Walls.
- **Zero-Latency Lead Capture**: Seamless integration with **EmailJS** for real-time inquiry transmission directly from the UI.
- **Immersive Motion Architecture**: Utilizes Intersection Observer APIs for scroll-triggered reveal animations and hardware-accelerated CSS marquee systems.

---

## 🛠️ The Tech Stack

### Frontend Engineering
- **Vanilla JavaScript (ES6+)**: Custom logic for state management, sliders, and the scope calculator.
- **Advanced CSS3 Architecture**: Extensive use of Flexbox, Grid, and custom variables for a responsive, high-contrast dark-mode aesthetic.
- **HTML5**: Semantic structure optimized for SEO and accessibility.

### Integration & Performance
- **EmailJS SDK**: Asynchronous form handling for reliable lead generation.
- **FontAwesome & Google Fonts**: Curated typography and iconography for premium brand positioning.
- **Vite/Local Dev**: Optimized asset delivery and performance-first image loading.

---

## 📂 System Highlights

- **`comparison-slider`**: A custom UI component handling mouse and touch events for visual data comparison.
- **`Project Scope Estimator`**: Algorithmic pricing logic based on acreage and architectural complexity.
- **`Lightbox System`**: A reusable modal architecture for high-resolution project documentation.

## 🚀 Deployment & Local Setup

1. **Clone the Asset Base**
   ```bash
   git clone https://github.com/DTE-84/TonysLandscaping.git
   cd TonysLandscaping
   ```

2. **Initialize Service Keys**
   Ensure your `EmailJS` public keys are correctly mapped in `script.js` to enable lead capture functionality.

3. **Launch Terminal**
   Simply open `index.html` via Live Server or any static hosting protocol.
